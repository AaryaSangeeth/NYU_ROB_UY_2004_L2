# NYU ROB_UY_2004 Robotic Class

## Installation

```bash
conda create --name rob2004 python=3.11
conda activate rob2004
pip install -e .
```
## Simulation
Run the simulator located at `scripts/simulator_app.py`:

```bash
python scripts/simulator_app.py
```
**Important:** If you're simulating on a Mac, you must use `mjpython` to run the `scripts/simulator_app.py` script:

```bash
mjpython scripts/simulator_app.py
```
